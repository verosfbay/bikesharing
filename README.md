# Bikesharing Visualization

## Purpose
The purpose of this visualization is to convince investors that a bike-sharing program in Des Moines is a solid business proposal. To solidify the proposal, one of the key stakeholders would like to see a bike trip analysis. I will analyze the data using Tableau Public. 

### Tools Used
* Tableau Public
* Jupyter Notebook
* Pandas

[link to dashboard](https://public.tableau.com/views/Bikesharingchallenge_16561810030240/BikeshareStory?:language=en-US&:display_count=n&:origin=viz_share_link)
